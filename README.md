# Sales Prediction using Python

## Overview

This project predicts product sales based on advertising expenditure across TV, Radio, and Newspaper channels.

The objective is to identify which advertising channels contribute most to sales and build a predictive model for future business decisions.

## Dataset

Source: Kaggle Advertising Dataset

Features:

- TV
- Radio
- Newspaper

Target Variable:

- Sales

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Building
6. Model Evaluation
7. Business Insights

## Model Used

- Linear Regression

## Evaluation Metrics

- MAE
- RMSE
- R² Score

## Results

The model achieved an R² score of approximately 0.90.

## Conclusion

TV and Radio advertising have a stronger impact on sales compared to Newspaper advertising.

Businesses can optimize their advertising budgets by focusing on high-performing channels.
